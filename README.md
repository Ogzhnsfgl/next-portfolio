## Portfolia Site Practice with NextJS

It's a portfolio website practice for learning and warming up Next JS.

Live demo available here: [ <b> Live Demo</b> ](https://next-portfolio-seven-flame.vercel.app/)

## Technologies Used

- NextJS
- Chakra-UI
- React-Icons
- Emotion-React
- Emotion-Styled
- Framer-Motion
- ThreeJS
